# Gatekeeper Docs

This folder contains the public documentation for Gatekeeper, a Discord bot that helps guilds run fair and consistent member trials.

## What this documentation covers

- What Gatekeeper does and when to use it.
- How to install and configure it in a Discord server.
- Command references for trial lifecycle, feedback, and administration.
- Recommended workflows for consistent officer decisions.

## Main docs sections

- [Getting Started](src/content/docs/getting-started/overview.md)
- [Commands](src/content/docs/commands/index.mdx)
- [Workflows](src/content/docs/workflows/trial-lifecycle.md)
- [FAQ](src/content/docs/faq.md)
- [Changelog](src/content/docs/changelog.md)

## Run docs locally

From the repository root:

- `pnpm docs:dev` to run the docs site.
- `pnpm docs:build` to build the docs.
- `pnpm docs:check` to run Astro checks.
