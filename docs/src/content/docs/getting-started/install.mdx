## Post-install checklist

1. Confirm Gatekeeper appears in your member list.
2. Run `/ping` as an officer to validate command visibility.
3. Run `/settings` and save officer channel, trial role, raider role, and schedule values.
4. Run `/list` to confirm the bot can read and respond ephemerally.
5. Run a test cycle using the [First Trial Walkthrough](./first-trial/).
