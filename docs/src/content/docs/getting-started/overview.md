Gatekeeper is a Discord bot for structured trial management in guilds and teams.

## Core capabilities

- Start and resolve trials with clear state transitions.
- Collect standardized officer feedback with score fields and comments.
- Generate summaries and run officer-channel voting.
- Configure reminders based on raid scheduling and attendance thresholds.
- Keep all records scoped by guild to avoid cross-server data overlap.

## Who should use it

- Raid teams onboarding new members.
- Officer groups that need consistent promotion decisions.
- Communities that want repeatable workflows instead of ad hoc chat messages.
